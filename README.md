# Facebook Irc Gateway
おなかすいた

## つかうまえに
Facebook のアプリを申請しましょう。

  http://www.facebook.com/developers/

アプリ ID と秘訣を使って、 CODE を取得しましょう。

  http://developers.facebook.com/docs/authentication/

## つかいかた
config.yaml を設定する

起動する

  ./fig.rb

IRC クライアントで繋ぐ。 (デフォルトだと 16822 あたりのポートを Listen する)
